How to read the Graph

Each node in the graph represents a streamer, while the links between nodes indicate the chatter in common between the connected streamers. In other words, if two nodes are connected by an arc, it means that they share a portion of their viewer audience.

The use of colors to distinguish the various Twitch communities provides a clear view of the affinities and connections within the network of Italian streamers.

Tecnical Description

To determine the relative connectedness between streamers, a hybrid approach was taken that takes into account both the number of shared users and the number of messages sent by those shared users. This method involved creating a detailed list of each channel's chatters, with each chatter represented as many times as the number of hours it participated in the streamer's chat. These lists were then intersected to assess common interactions between streamers.

For the creation of the graph, Gephi software was used. In particular, the Force Atlas 2 spatialization algorithm was applied, which allowed the nodes of the graph to be positioned in such a way as to maximize the clarity of the relationships between streamers. This approach made possible a dynamic and intuitive visualization of the complex network of interactions among Italian streamers on Twitch.
